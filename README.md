jQuery extensions plugin
====================

## Extensions methods:

##### Resize to max-width/height
```javascript
$(el).resizeToMaxWidth();
```

##### Get array of heights
```javascript
var heightsMap = $(el).heightsMap();
```

##### Ensure visible
```javascript
$.ensureVisible(selector, options);
```
